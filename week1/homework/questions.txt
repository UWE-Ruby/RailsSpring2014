Questions:
1. What is the difference between Bundler and RVM?
	RVM manages the Ruby/Gem environment - which versions to use.  Bundler handles dependencies between gems.
2. What is REST and why is it useful?
	REST organizes comunication between system components using routes.  It uses consistent naming and organization for apps.
3. What does CRUD stand for? 
	Create, Read, Update, Destroy
4. What does this line of code do, and what file in my rails app would it go in?
   resources :students, except: :destroy

   It creates all routes for Student except for destroy.
