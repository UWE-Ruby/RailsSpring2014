Questions:
1. What is the difference between Bundler and RVM?
Bundler manages gems and RVM manages versions of Ruby.

2. What is REST and why is it useful?
A network architectural style manifested in the architecture of the World Wide Web. It is useful because it provides standardization of routing commands.

3. What does CRUD stand for? 
Create Read Update Delete

4. What does this line of code do, and what file in my rails app would it go in?
   resources :students, except: :destroy
It creates the standard routes for students, except for destroy. It is placed in the routes.rb file.
